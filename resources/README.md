# 🗂️ Resources

Shared, cross-cutting references — the material that serves *every* design rather than belonging to one. Where a `designs/NN-*` folder answers "how do I build X?", this folder answers the questions that recur across all of them.

| Resource | What it is |
|----------|------------|
| [📐 Estimation](./estimation/) | Back-of-the-envelope calculation of QPS, storage, bandwidth, latency and availability — the toolkit for **step 4 of every design**. Includes a reference doc, a recall deck, a printable wall poster, and an interactive calculator. |

_More will land here as patterns repeat across designs — consistency, replication, caching strategies, and the like._
